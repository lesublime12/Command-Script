# Command Script (CSC)
Scripting language inspired by CoD WaW commands.

## Commands
|      Name     | Description |
| ------------- | ----------- |
|         bind | key |
|       unbind | key |
|    unbindall | key |
|       prompt | input |
|          set | variable |
| setvartotime | variable |
|        unset | variable |
|    increment | math |
|          add | math |
|     subtract | math |
|     multiply | math |
|        pvstr | condition |
|        zvstr | condition |
|        nvstr | condition |
|         vstr | function-like |
|       toggle | variable |
|      protect | interpret | (vars cannot be overwritten by program after protect run, etc)
|      cmdlist | output |
|      varlist | output |
|         wait | wait |
|         echo | output |
|        clear | output |

## Variables
|   Name   | Description |
| -------- | ----------- |
|  iv_time | epoch |
| iv_error | error code |
|  iv_mode | interpret mode (protected/user) |
